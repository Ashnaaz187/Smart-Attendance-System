## 🚀 Overview

An **AI-powered Smart Attendance System** that automates attendance using **real-time face detection, recognition, and tracking**.

Built using advanced computer vision models like **YOLOv8, FaceNet, and DeepSORT**, this system ensures:
- Accurate identification  
- Duplicate prevention  
- Real-time analytics  

---

## ✨ Features

✔ Real-time face detection using **YOLOv8**  
✔ Face recognition using **FaceNet embeddings**  
✔ Multi-person tracking using **DeepSORT**  
✔ Automatic attendance marking  
✔ Duplicate entry prevention  
✔ 📊 Premium dashboard with analytics  
✔ 🟩 GitHub-style heatmap visualization  
✔ 📥 Downloadable attendance reports (CSV)  
✔ 📧 Email alerts for attendance
