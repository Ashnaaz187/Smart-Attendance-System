# 🎯 Smart Attendance System

An AI-based Smart Attendance System that automates attendance using face recognition and tracking.

## 🚀 Features
- Real-time face detection using YOLO
- Face recognition using FaceNet
- Multi-person tracking using DeepSORT
- Automatic attendance marking with duplicate prevention
- Interactive dashboard with analytics & heatmap
- Email alerts for attendance updates

## 🧠 Tech Stack
- Python
- OpenCV
- YOLO (Ultralytics)
- FaceNet
- DeepSORT
- Streamlit
- SQLite
- Pandas, Plotly, Seaborn

## 📊 Dashboard Includes
- Daily attendance trends
- Person-wise analysis
- GitHub-style heatmap
- Downloadable reports
